Selection Sort



Selection Sort is another simple way to sort a list of items, like numbers or words. It works by looking through the entire list to find the smallest (or largest) item. Once the smallest item is found, it gets swapped with the first item in the list. Then, the process is repeated for the rest of the list, starting from the second item. The next smallest item is found and swapped with the second position, and so on, until the entire list is sorted. The main idea is that each time you go through the list, you "select" the smallest item and put it in the correct place. While it's easy to understand, Selection Sort can be slow for large lists.
