Pseudocode

What is Pseudocode?

Pseudocode is called false code because it resembles a programming language but is easy to understand even for someone with little programming knowledge. It is a method for representing algorithms in a programming-like format helping programmers outline their logic before actual coding.

Five Examples of Pseudocode

1: Finding the Largest Number in a Listt

    START
      INPUT list of numbers
      SET largest = first number in the list
      FOR each number in the list
        IF number > largest
            SET largest = number
        ENDIF
      ENDFOR
    PRINT largest
    END

2: Calculating Factorial of a Number

    START
      INPUT number
      SET factorial = 1
      FOR i = 1 TO number
         SET factorial = factorial * i
      ENDFOR
    PRINT factorial
    END

3: Linear Search

    START
      INPUT list of numbers, target
      FOR each number in the list
        IF number == target
         PRINT "Found"
         STOP
         ENDIF
      ENDFOR
    PRINT "Not Found"
    END

4: Checking if a Number is Prime

    START
      INPUT number
      SET isPrime = true
      FOR i = 2 TO number-1
        IF number % i == 0
          SET isPrime = false
          BREAK
        ENDIF
      ENDFOR
    IF isPrime
      PRINT "Number is Prime"
    ELSE
      PRINT "Number is not Prime"
    END

5: Reversing a String

     START
      NPUT string
      SET reversed = ""
      FOR i = length of string TO 1
        SET reversed = reversed + character at position i
      ENDFOR
    PRINT reversed
    END
